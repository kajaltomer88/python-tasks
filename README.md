Task PY200_T2 - Number Check (Positive, Negative or Zero)

📌 Description
This Python program checks whether the entered number is:

Positive
Negative
Zero
If the number is negative, the program also displays its positive value using formatted string printing.

💻 Concepts Used
User Input
Conditional Statements (if-elif-else)
Absolute Function abs()
Formatted String Printing (f-string)
Type Handling using float() and is_integer()
⚙ How It Works
The program takes numeric input from the user.
It converts the input into float.
If the number is an integer value (like 4.0), it converts it to int for clean display.
It checks:
If number > 0 → prints positive message
If number < 0 → prints negative message and its positive equivalent
If number == 0 → prints zero message
▶ Example Run
Input:-7 Your entered number -7 was negative, however positive value of your number is 7

🎯 Learning Outcome
This task demonstrates the use of:

Conditional logic
Basic mathematical operations
Clean output formatting in Python
